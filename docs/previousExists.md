# $previousExists
Check whether a previous track exists or not.
## Usage
```
$previousExists[guildID]
```
## Fields
|   Name   |     Description      | Type  | Required | Rest |
|----------|----------------------|-------|----------|------|
| Guild ID | The ID of the player | Guild | Yes      | No   |

## Output
> Boolean
