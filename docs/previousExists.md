# $previousExists
Check whether a previous track exists in the player queue.
## Usage
```
$previousExists[guildID]
```
## Fields
|   Name   |                 Description                  | Type  | Required | Rest |
|----------|----------------------------------------------|-------|----------|------|
| Guild ID | The ID of the guild to check the player for. | Guild | Yes      | No   |

## Output
> Boolean
