# $playerElapsedTime
Returns the elapsed time of the current song in milliseconds.
## Usage
```
$playerElapsedTime[guildID]
```
## Fields
|   Name   |     Description      | Type  | Required | Rest |
|----------|----------------------|-------|----------|------|
| Guild ID | The ID of the guild  | Guild | Yes      | No   |

## Output
> String
