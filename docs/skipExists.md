# $skipExists
Check whether the next track exists in the player queue.
## Usage
```
$skipExists[guildID]
```
## Fields
|   Name   |                 Description                  | Type  | Required | Rest |
|----------|----------------------------------------------|-------|----------|------|
| Guild ID | The ID of the guild to check the player for. | Guild | Yes      | No   |

## Output
> Boolean
