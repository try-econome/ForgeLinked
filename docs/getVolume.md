# $getVolume
gets the guild player volume/loudness
## Usage
```
$getVolume[guildID]
```
## Fields
|   Name   |     Description     | Type  | Required | Rest |
|----------|---------------------|-------|----------|------|
| Guild ID | The ID of the guild | Guild | Yes      | No   |

## Output
> String
