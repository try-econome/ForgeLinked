# $queueHistory
Displays the history of previously played tracks from the guild player.
## Usage
```
$queueHistory[guildID]
```
## Fields
|   Name   |     Description     | Type  | Required | Rest |
|----------|---------------------|-------|----------|------|
| Guild ID | The ID of the guild | Guild | Yes      | No   |

## Output
> Json
