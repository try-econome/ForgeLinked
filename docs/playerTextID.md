# $playerTextID
gets the players text channel id.
## Usage
```
$playerTextID[guildID]
```
## Fields
|   Name   |                 Description                 | Type  | Required | Rest |
|----------|---------------------------------------------|-------|----------|------|
| Guild ID | The ID of the guild get the player ID from. | Guild | Yes      | No   |

## Output
> TextChannel
