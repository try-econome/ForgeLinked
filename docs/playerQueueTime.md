# $playerQueueTime
Returns the total duration of all tracks in the player queue in milliseconds.
## Usage
```
$playerQueueTime[guildID]
```
## Fields
|   Name   |     Description      | Type  | Required | Rest |
|----------|----------------------|-------|----------|------|
| Guild ID | The ID of the guild  | Guild | Yes      | No   |

## Output
> String
