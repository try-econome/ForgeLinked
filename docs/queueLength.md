# $queueLength
displays the queue length in a specific guild
## Usage
```
$queueLength[guildID]
```
## Fields
|   Name   |     Description      | Type  | Required | Rest |
|----------|----------------------|-------|----------|------|
| Guild ID | The ID of the guild  | Guild | Yes      | No   |

## Output
> String
