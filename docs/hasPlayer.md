# $hasPlayer
Check whether the given guild has a player created.
## Usage
```
$hasPlayer[guildID]
```
## Fields
|   Name   |                 Description                  | Type  | Required | Rest |
|----------|----------------------------------------------|-------|----------|------|
| Guild ID | The ID of the guild to create the player to. | Guild | Yes      | No   |

## Output
> Boolean
