# $setVolume
Sets the guild player volume/loudness (recomended limit:200) 
## Usage
```
$setVolume[guildID;volume]
```
## Fields
|   Name   |     Description     |  Type  | Required | Rest |
|----------|---------------------|--------|----------|------|
| Guild ID | The ID of the guild | Guild  | Yes      | No   |
| volume   | volume              | Number | Yes      | No   |

## Output
> String
