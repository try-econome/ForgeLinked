# $clearQueue
Clears the queue/all tracks in a specific guild
## Usage
```
$clearQueue[guildID]
```
## Fields
|   Name   |     Description      | Type  | Required | Rest |
|----------|----------------------|-------|----------|------|
| Guild ID | The ID of the guild  | Guild | Yes      | No   |

## Output
> String
