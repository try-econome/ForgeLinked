# $queue
displays the queue/all tracks from the guild player
## Usage
```
$queue[guildID]
```
## Fields
|   Name   |     Description      | Type  | Required | Rest |
|----------|----------------------|-------|----------|------|
| Guild ID | The ID of the guild  | Guild | Yes      | No   |

## Output
> Json
