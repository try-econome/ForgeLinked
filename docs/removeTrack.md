# $removeTrack
Removes a track from the guild player
## Usage
```
$removeTrack[guildID;position]
```
## Fields
|   Name   |     Description     |  Type  | Required | Rest |
|----------|---------------------|--------|----------|------|
| Guild ID | The ID of the guild | Guild  | Yes      | No   |
| Position | track position      | Number | Yes      | No   |

## Output
> String
