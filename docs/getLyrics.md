# $getLyrics
Fetches lyrics for the currently playing track in the guild.
## Usage
```
$getLyrics[guildID]
```
## Fields
|   Name   |                Description                | Type  | Required | Rest |
|----------|-------------------------------------------|-------|----------|------|
| Guild ID | The ID of the guild to fetch lyrics from. | Guild | Yes      | No   |

## Output
> String
