# $searchEngine
Sets the player's search engine.
## Usage
```
$searchEngine[engine]
```
## Fields
|  Name  |                    Description                    |  Type  | Required | Rest |
|--------|---------------------------------------------------|--------|----------|------|
| Engine | The engine to use (ytsearch, scsearch, spsearch). | String | Yes      | No   |

## Output
> Json
