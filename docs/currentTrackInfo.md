# $currentTrackInfo
Gets info on a track in a specific guild
## Usage
```
$currentTrackInfo[guildID]
```
## Fields
|   Name   |     Description     | Type  | Required | Rest |
|----------|---------------------|-------|----------|------|
| Guild ID | The ID of the guild | Guild | Yes      | No   |

## Output
> Json
